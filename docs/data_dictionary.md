# Data Dictionary

| Variable | Description |
|---|---|
| listing_id | Unique listing identifier |
| district | Lyon district name |
| district_number | Numerical arrondissement identifier |
| rooms | Number of rooms |
| bedrooms | Number of bedrooms |
| area_sqm | Property area in square meters |
| price | Listing price in euros |
| price_per_sqm | Price divided by area |
| housing_type | Housing category based on room count |

---

## Housing Type Categories

| Category | Definition |
|---|---|
| Studio/T1 | 1 room |
| T2 | 2 rooms |
| T3 | 3 rooms |
| T4+ | 4 or more rooms |
